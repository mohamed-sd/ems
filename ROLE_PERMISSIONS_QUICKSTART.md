# 🚀 دليل البدء السريع - Role Permissions

## ⚡ في 3 خطوات فقط

### 1️⃣ الإعداد (Setup)
```
1. ذهب إلى: /setup_role_permissions.php
2. انقر: "إنشاء الجدول الآن"
3. انتظر رسالة النجاح ✅
```

### 2️⃣ الوصول إلى الشاشة
```
1. ذهب إلى: Settings > الإعدادات
2. انقر: "⭐ إدارة الصلاحيات المفصلة"
3. الآن أنت في صفحة الإدارة! 🎉
```

### 3️⃣ إضافة صلاحيات
```
الطريقة أ - الشبكة (الأسهل):
1. اختر دور من القائمة
2. حدد الصلاحيات ☑️
3. انقر "حفظ"

الطريقة ب - السريعة (الأسرع):
1. انقر "الإجراءات السريعة"
2. اختر دور
3. انقر "منح الكل" أو "سحب الكل"
```

## 🎯 أمثلة سريعة

### مثال 1: إعطاء دور الوصول الكامل
```
1. اختر الدور
2. انقر الإجراءات السريعة
3. انقر "منح الكل"
✅ يمكنه الآن فعل كل شيء!
```

### مثال 2: حذف صلاحية واحدة
```
1. انقر عرض الجدول
2. جد الصف المطلوب
3. انقر الزر الأحمر (حذف)
❌ تم إزالة الصلاحية!
```
