# إدارة الصلاحيات والأدوار - نقل إلى الإدارة العليا

## 📋 ملخص التغييرات

تم نقل نظام إدارة الصلاحيات والأدوار من `/Settings/settings.php` إلى الإدارة العليا `/admin/permissions/` بحيث أصبح متاحاً فقط للسوبر ادمن.

## 📁 الملفات الجديدة

### `/admin/permissions/index.php`
صفحة رئيسية توضح جميع خيارات إدارة الصلاحيات والأدوار بشكل منظم وسهل الاستخدام.

**يحتوي على:**
- عرض بطاقات جميلة لكل خيار (أدوار، صفحات، صلاحيات)
- وصف كامل لكل خيار
- روابط سريعة إلى كل صفحة

### `/admin/permissions/roles.php`
إدارة الأدوار والرتب في النظام

**الميزات:**
- إنشاء أدوار جديدة
- تعديل الأدوار الموجودة
- تعيين الهرمية بين الأدوار (دور أب)
- تفعيل/تعطيل الأدوار
- حذف الأدوار غير المستخدمة
- عرض جميع الأدوار في جدول DataTable

### `/admin/permissions/modules.php`
إدارة الصفحات والمديولات

**الميزات:**
- إضافة صفحات جديدة للنظام
- ربط كل صفحة بدور مسؤول
- اختيار أيقونة لكل صفحة
- تحديد ما إذا كانت الصفحة رابط أم لا
- فلترة الصفحات حسب الدور المسؤول
- عرض جميع الصفحات في جدول

### `/admin/permissions/role_permissions.php`
إدارة الصلاحيات المفصلة

**الميزات:**
- اختيار دور معين
- عرض جميع الصفحات المسندة لهذا الدور
- منح صلاحيات محددة (عرض، إضافة، تعديل، حذف) لكل صفحة
- إجراءات سريعة: منح جميع الصلاحيات أو سحبها
- عرض إحصائيات عن الأدوار والصفحات والصلاحيات
- عرض جدول شامل لجميع الصلاحيات

## 🔒 التحقق من الوصول

جميع الملفات الجديدة محمية بـ:
```php
require_once __DIR__ . '/../includes/auth.php';
super_admin_require_login();
```

هذا يعني أن **فقط السوبر ادمن** يمكنه الوصول إلى هذه الصفحات.

## 🎯 روابط الملاح

تم إضافة رابط جديد في الملاح الرئيسي `/admin/includes/layout_head.php`:

```
إدارة الصلاحيات → /admin/permissions/
```

الرابط:
- **العنوان:** إدارة الصلاحيات
- **الأيقونة:** fa-lock-open
- **الموقع:** بين "إدارة الشركات" و "طلبات الاشتراك"

## 🔄 المسارات النسبية

تم تحديث جميع المسارات النسبية:
- من `../` (Settings) إلى `../` (admin/permissions)
- استخدام `include '../config.php'` للاتصال بقاعدة البيانات
- استخدام `require_once __DIR__ . '/../includes/layout_head.php'` للـ header

## 📊 البنية

```
/admin/
├── permissions/
│   ├── index.php                 # الصفحة الرئيسية
│   ├── roles.php                 # إدارة الأدوار
│   ├── modules.php               # إدارة الصفحات
│   └── role_permissions.php      # إدارة الصلاحيات التفصيلية
└── includes/
    ├── layout_head.php           # تم تحديثه بإضافة رابط الصلاحيات
    ├── auth.php
    └── layout_foot.php
```

## 🎨 التصميم

جميع الصفحات تتبع نفس تصميم الإدارة العليا:
- نفس الألوان والأسلوب
- نفس DataTables و Bootstrap
- نفس Font Awesome للأيقونات
- واجهة عربية RTL كاملة

## ⚠️ ملاحظات مهمة

1. **الملفات الأصلية:** ملفات `/Settings/` الأصلية بقيت كما هي للآن (للانتقال السلس)
2. **قاعدة البيانات:** لا توجد تغييرات على قاعدة البيانات
3. **الصلاحيات:** عملية الانتقال لا تؤثر على النظام القديم
4. **الدعم:** جميع الوظائف محفوظة بالكامل

## 🚀 الخطوات التالية (اختيارية)

بعد التأكد من عمل النظام الجديد، يمكنك:

1. **إعادة توجيه الرابط القديم** من `/Settings/settings.php` إلى `/admin/` إذا رغبت
2. **حذف الملفات القديمة** من `/Settings/` بعد التأكد من عدم استخدامها
3. **تحديث أي روابط** تشير مباشرة إلى الملفات القديمة

## 📞 الدعم

للحصول على مساعدة أو الإبلاغ عن مشاكل، يرجى التواصل مع الدعم الفني.
